## Training Git

This is project introduce to git

## Installation

### Git
Go to [git](https://git-scm.com/) then [download](https://git-scm.com/downloads) and install

### Git Hub
Go to [github](https://github.com) then create account

### Git CLI
	
	git --version

or

	git -v
	
#### Git Status
		
	git status
	
#### Git Add
		
	git add file-name
	git add .
	git add *.txt
	
#### Git Commit

	git commit -m your-messages
